# Finsyt

AI-powered financial intelligence workspace. **Next.js 15 App Router · TypeScript · Tailwind CSS**

## Routes

| Route | Page |
|-------|------|
| `/` | Landing — hero, features, CTA |
| `/pricing` | Pricing — Free / Pro / Team |
| `/dashboard` | Dashboard — portfolio, signals, AI insights |
| `/auth/sign-in` | Sign In |
| `/auth/sign-up` | Sign Up |
| `/settings` | Settings — profile, notifications, billing |

## Quick start

```bash
cp .env.example .env.local
npm install
npm run dev
```

## Deploy

Push to GitHub → import at [vercel.com/new](https://vercel.com/new) → auto-detect Next.js → Deploy.
